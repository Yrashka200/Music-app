# Music App

A simple browser-based music player built with HTML, CSS, and JavaScript.

## Overview

This project is inspired by the idea from https://github.com/kaizhelam/Music-Player and extends it with a custom layout, track controls, and album art support.

## Features

- Play, pause, next, and previous track controls
- Seek bar with current time and duration display
- Volume control slider
- Random track mode
- Album art display
- Simple responsive design

## Usage

1. Open `index.html` in your browser.
2. Add audio files to the `music/` folder.
3. Add matching cover images to the `images/` folder.
4. Update `script.js` to include new tracks in the `music_list` array.
